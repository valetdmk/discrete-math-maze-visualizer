# Визуализатор лабиринта — DFS + BFS

Учебный проект по алгоритмам на графах. Генерирует лабиринт алгоритмом DFS и ищет кратчайший путь через BFS с пошаговой анимацией.
Выполнен в группе из 
Колеватов Г.А - BE ( и чуть чуть FE, по работе с canvas ) 
Качура К.М - FE

Посмотреть работу можно здесь: https://valetdmk.github.io/maze_visualizer/
## Запуск

Открыть файл `maze_visualizer.html` в браузере — никаких зависимостей и сервера не требуется.

## Возможности

- **Генерация лабиринта** — DFS recursive backtracking, случайный лабиринт при каждом нажатии
- **Поиск пути** — BFS от старта (S) до финиша (F), гарантирует кратчайший путь
- **Анимация** — сначала подсвечиваются все посещённые BFS клетки, затем восстанавливается путь
- **Настройки** — размер сетки от 5×5 до 31×31, скорость анимации ×1–×5
- **Статистика** — количество посещённых клеток, длина пути, шагов BFS
- **Экспорт JSON** — сохраняет все шаги обхода в файл `bfs_steps.json`

## Алгоритмы

**DFS (генерация)** — рекурсивный backtracking:
1. Начать с клетки (1,1), пометить как проход
2. Выбрать случайного непосещённого соседа через одну клетку
3. Пробить стену между текущей и соседней клеткой
4. Рекурсивно повторить для соседа; при тупике — откатиться

**BFS (поиск пути)**:
1. Добавить старт в очередь
2. Для каждой клетки из очереди проверить 4 соседей
3. Непосещённые проходимые соседи — добавить в очередь, записать родителя
4. При достижении финиша — восстановить путь по таблице родителей

## Формат экспорта JSON

```json
{
  "maze": { "rows": 15, "cols": 15 },
  "bfs": {
    "totalSteps": 87,
    "pathLength": 24,
    "steps": [
      { "step": 0, "row": 1, "col": 1, "type": "visit", "cell": "start" },
      { "step": 1, "row": 1, "col": 2, "type": "visit", "cell": "path" },
      ...
      { "step": 110, "row": 13, "col": 13, "type": "path" }
    ]
  }
}
```

Поле `type`: `visit` — клетка посещена BFS, `path` — клетка входит в кратчайший путь.
